# Use the iNsidr DevTools in tour app  

Include in Your HTML  

```html
<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="path/to/devtools.css">
</head>
<body>
    <!-- Your content -->

    <script src="path/to/devtools.bundle.js"></script>
    <script>
        window.CustomDevTools.init();
    </script>
</body>
</html>
```
